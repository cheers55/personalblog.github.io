    # 🌈 学习 Vue 源码

    # 1、Object.defineProperty()

    Object.defineProperty(obj, prop, desc)的作用就是直接在一个对象上定义一个新属性，或者修改一个已经存在的属性。

    1. obj 需要定义属性的当前对象；
    2. prop 当前需要定义的属性名；
    3. desc 属性描述符；
    const obj = {};
let a;
Object.defineProperty(obj, 'a', {
    get(){
        console.log('get val');
        return a;
    },
    set(newVal){
        console.log('set val:' + newVal);
        a = newVal;
    }
});
obj.a; // get val
obj.a = '666'; // set val: 666
