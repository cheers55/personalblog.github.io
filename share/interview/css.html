        # 3. 双飞翼布局

        主要解决的实际问题是：定制化展示内容

        例如：优先渲染中间内容，因为中间内容更重要。

        // order的主要作用是定制化展示顺序
.box {
    display: flex;
}

.center {
    flex: 1;
    order: 2;
    background: red;
}

.left, .right {
    width: 200px;
}

.left {
    order: 1;
    background: green;
}

.right {
    order: 3;
    background: pink;
}
<div class="box">
    <div class="center">
        <div>中间内容</div>
    </div>
    <div class="left">左边内容</div>
    <div class="right">右边内容</div>
</div>
