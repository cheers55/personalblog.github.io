# 🌈 算法面试题总结

# 1. 深度优先遍历 & 广度优先遍历

需求：在页面中某个 dom 节点中遍历，找到目标 dom 节点。

区别：

  1. 深度优先不需要记住全部节点占用空间小，广度优先必须要先记录全部的节点占用空间大；
  2. 深度优先有回溯的操作相对时间更长；
  3. 深度优先采用堆栈的形式，即先进后出。广度优先采用队列的形式，即先进先出；
  • 假设页面上 dom 结构如下：

            <a href="">
                <img src="">
            </a>
        </li>
        <li>
            <span>
            </span>
        </li>
        <li></li>
    </ul>

    <p></p>

    <button></button>
</div>

# 1.1 深度优先遍历（Depth-First Search）

以纵向的维度对 dom 树进行遍历，从根节点开始，一直遍历其子节点，直到它的所有子节点都被遍历完之后再遍历它的兄弟节点。

function dfs(node, nodeList){
    if(node){
        nodeList.push(node);
        const children = node.children;
        for(let i = 0; i < children.length; i++>){
            dfs(children[i], nodeList);
        }
    }

    return nodeList;
}

const root = document.getElementById('root');
dfs(root, nodeList = []);

function dfs(node){
    const nodeList = [];
    if(node != null){
        const stack = [];
        stack.push(node);
        while(stack.length != 0){
            const item = stack.pop();
            nodeList.push(item);
            const children = item.children;
            for(let i = children.length - 1; i>=0; i--){
                stack.push(children[i]);
            }
        }
    }

    return nodeList;
}

# 1.2 广度优先遍历（breadth-first traverse）

以横向的维度对 dom 树进行遍历，从根节点开始，从左往右遍历其所有的兄弟节点，以此类推。

function bfs(node){
    const nodeList = [];
    if(node != null){
        const queue = [];
        queue.unshift(node);
        while(queue.length != 0){
            const item = queue.shift();
            nodeList.push(item);
            const children = item.children;
            for(let i = 0; i < children.length; i++>){
                queue.push(children[i]);
            }
        }
    }

    return nodeList;
}

const root = document.getElementById('root');
bfs(root);
