        # 1. http1.0、http1.1、http2.0、http3.0 的区别

        类型 http1.0 http1.1 http2.0 http3.0
        连接 默认非持久连接 持久连接，多个 http 请求复用同一个 TCP 连接 xxx xxx
        资源请求 不支持断点续传功能 支持 xxx xxx
        缓存 If-Modified-Since、Expires Etag、If- xxx xxx

        # 2. GET 和 POST 的区别

        类型 GET POST
        请求长度 浏览器对 url 长度有限制 无限制
        安全性 由于参数暴露在 url 上，不太安全 相对安全
        是否缓存 浏览器一般会缓存 不缓存
        请求速度 相对较快 相对较慢，第一次请求待返回 100 后，再次发送请求
